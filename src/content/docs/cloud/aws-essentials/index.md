**Everything you need to be productive on AWS — before touching EKS.**

Not everything goes into Kubernetes. This track covers the core AWS services every DevOps engineer needs: identity, networking, compute, containers, serverless, secrets, observability, CI/CD, and infrastructure as code.

---

## Modules

| # | Module | Time | What You'll Learn |
|---|--------|------|-------------------|
| 1 | [IAM Deep Dive](module-1-iam.md) | 2h | Roles, policies, cross-account, permission boundaries |
| 2 | [VPC & Core Networking](module-2-vpc.md) | 3h | Subnets, NAT, Security Groups, Transit Gateway |
| 3 | [EC2 & Compute](module-3-ec2.md) | 2.5h | Instance types, Spot, ASGs, ALBs |
| 4 | [S3 & Object Storage](module-4-s3.md) | 1.5h | Storage classes, lifecycle, bucket policies |
| 5 | [Route 53 & DNS](module-5-route53.md) | 1.5h | Routing policies, health checks, DNSSEC |
| 6 | [ECR (Container Registry)](module-6-ecr.md) | 1h | Image management, scanning, lifecycle |
| 7 | [ECS & Fargate](module-7-ecs-fargate.md) | 3h | Standalone containers without K8s |
| 8 | [Lambda & Serverless](module-8-lambda.md) | 2h | Triggers, cold starts, Step Functions |
| 9 | [Secrets Management](module-9-secrets.md) | 1.5h | SSM, Secrets Manager, KMS |
| 10 | [CloudWatch & Observability](module-10-cloudwatch.md) | 2h | Metrics, alarms, logs, X-Ray |
| 11 | [CI/CD on AWS](module-11-cicd.md) | 2h | CodePipeline, CodeBuild, GitHub Actions OIDC |
| 12 | [CloudFormation](module-12-cloudformation.md) | 1.5h | Templates, stacks, CDK overview |

**Total time**: ~23 hours
