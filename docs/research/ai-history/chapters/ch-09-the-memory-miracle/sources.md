## Conflict Notes

- **Wang / Way-Dong Woo co-inventorship.** Patent 2,708,722 lists An Wang as sole inventor; multiple secondary sources name Way-Dong Woo as co-inventor working alongside Wang at Harvard's Computation Laboratory.
