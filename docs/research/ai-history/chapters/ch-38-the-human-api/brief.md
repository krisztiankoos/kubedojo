# Brief: Chapter 38 - The Human API

## Thesis
Machine learning requires massive, annotated datasets. When algorithms couldn't classify data, Amazon created Mechanical Turk, transforming human cognitive labor into an API endpoint. This infrastructure allowed researchers to cheaply crowdsource the labels that would eventually train AI to replace that very labor.

## Scope
- IN SCOPE: Amazon Mechanical Turk (MTurk), "Artificial Artificial Intelligence", the economics of micro-labor, crowdsourced data annotation.
- OUT OF SCOPE: Modern RLHF click-farms (belongs to Part 8/9).

## Scenes Outline
1. **The Duplicate Catalog:** Amazon struggles to find duplicate product listings using algorithms.
2. **Artificial Artificial Intelligence:** Jeff Bezos and team realize they can expose human workers as a web service API to do the tasks computers can't.
3. **The Annotation Engine:** AI researchers realize MTurk is the missing infrastructure for building massive, labeled training sets (the precursor to ImageNet).
