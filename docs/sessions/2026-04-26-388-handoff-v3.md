### Sourcing standard

## Lessons added this session

1. **Atomic drain pattern** (memory `feedback_drain_dirties_tree.md`): pause batch (`kill -INT`) before drain, OR accept-and-restart. Drain dirties tree → batch hits `dirty_pre`.
2. **Deferred-review rubber-stamp rate is ~94%, not 100%** as v2 implied. Real reviewer caught 2 modules (tracing, kubectl-basics) that otherwise would have shipped with quality gaps. Keep the deferred reviewer in the loop.
3. **Cherry-pick override on visual-aid regressions is sometimes correct** (kubectl-basics example: writer dropped a flat reference table but added a Worked Example + Did You Know — net pedagogical win) — but document the override in state.json AND don't skip the cross-family check.
4. **`pkill -f "dispatch.py codex"` is dangerous during an active batch** — it can hit the batch's in-progress writer. Always kill by exact PID.
5. **All three agents must explicitly AGREE on multi-step content plans** before files get written. Synthesis without their sign-off = not consensus.
6. **3-way discussion bridge has a claude-side bug** — monitor JSON snapshot in dispatch.py args breaks claude. Codex + gemini can converge without claude, but claude should not be silently absent. Open a bug, file with `scripts/ai_agent_bridge/_claude.py` reproduction.
