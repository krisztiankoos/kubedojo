Honest caveats

  • n=1 per cell on most data points. Codex variance (3/3 R1 → 1/4 R6 same prompt) demonstrates single-shot verdicts are signal, not ground truth. Multi-agent consensus reduces this risk.
  • Codex's "ground truth" on PR #1333 was itself incomplete — codex's own APPROVE missed the pip-pin issue 4/5 calibration candidates surfaced. So the deep claude-vs-codex comparison rests on a partially-flawed reference. Future calibrations should use multi-model consensus rather than one reviewer's verdict.
  • Some calibration cells aren't fair: codex caught F3 (cooldown) because it ran zizmor; candidates without tool-use received the diff text only. The lane "without tool-use codex still wins" is false — they're different test conditions.
  • Without HTTP probes, ALL models fabricate URL verification. Real URL verification requires codex gpt-5.5 + KUBEDOJO_CODEX_SEARCH=1. Every other model "verifies" via training-knowledge pattern-matching — sometimes right, sometimes confidently wrong.
  • Grok is removed from the roster as of session 29 (PR #1337). The architect-lane "Option C" strength was real but not enough to outweigh the reviewer-lane DQ pattern. Don't add back without re-calibration.
