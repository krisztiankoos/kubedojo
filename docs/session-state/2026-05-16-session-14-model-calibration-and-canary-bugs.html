Hermes-mediated Grok access (new tool pattern)

Grok-4.3 is reachable via hermes -z PROMPT -m grok-4.3 subprocess. Direct api.x.ai/v1 returns 403 on the OAuth-scoped token; subprocess is the only path. Hermes' default toolset is broad — for pure-text calibration use hermes -z "$PROMPT" -m grok-4.3 with prompt as a direct argument (NOT stdin via -; the dash is interpreted as a literal prompt).
