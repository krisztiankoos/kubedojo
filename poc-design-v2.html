Module 3.4: Ingress

⚠ Medium ⏱ 45-55 min 🏆 CKA Exam

Why This Module Matters

While Services expose applications to the network, Ingress provides HTTP/HTTPS routing with features like path-based routing, virtual hosts, and TLS termination.

⚠️ Important
The ingress-nginx controller reached end-of-life on March 31, 2026. The Ingress API itself is NOT deprecated.
💡 Did You Know?

The Gateway API took 4 years to reach GA (v1.0 in October 2023). The design required consensus from 8 different Ingress controller vendors.

Part 1: Ingress Architecture

1.1 How Ingress Works

An Ingress resource defines routing rules. An Ingress Controller watches for these resources and configures an actual load balancer accordingly.

yaml
apiVersion: networking.k8s.io/v1 kind: Ingress metadata: name: web-ingress spec: ingressClassName: nginx rules: - host: "app.example.com"
🔥 War Story: The Ingress That Brought Down Black Friday $2.3M lost

A major e-commerce platform used a single nginx Ingress controller for all traffic. On Black Friday, a misconfigured rate-limiting annotation caused 502s across all 47 services.

Lesson: Never share a single Ingress controller across environments.
